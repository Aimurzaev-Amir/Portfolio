Feedback form html + css + php + JavaScript + jQuery

By Amir Aimurzayev on 28 Feb 2020

Hello everyone, friends! In this article I want to share with you a simple feedback form for a site running on PHP, JavaScript and AJAX. This form has some additional functions, which are discussed in more detail later. You don’t need any special skills to create this form, just knowledge of the basics of html layout, CSS3 and elementary logic will be enough.

First of all, it is necessary to consider all the fields that will be available to us in this form.

The structure of the fields of the feedback form:

  • Sender name
  • Telephone
  • Mailing address
  • Customer message

Features of this form:

  • Inability to send messages without filling in all fields.
  • Lock the "Send message" button while sending a message.
  • Displays a success / failure window for sending a message.
  • Clearing the field after sending the message.

What does the feedback form include?

  • HTML markup (layout creation)
  • Styling form fields in CSS
  • JavaScript + AJAX
  • PHP handler

PHP code setup

Now let's move on to the form handler, which is written in PHP.

<?php
$name = $_POST['name'];
$phone = $_POST['phone'];
$email = $_POST['email'];
$message = $_POST['message'];

$subject = "=?utf-8?B?".base64_encode("Message from Front End Portfolio: aimurzayev.com")."?=";
$headers = "From: $email\r\nReply-to: $email\r\nContent-type: text/html; charset=utf-8\r\n";

$success = mail("aimurzaev@mail.ru", $subject, $message, $headers);
echo $success;
?>

Additional features in JavaScript and jQuery

Form field handler + sending a message without reloading the page (ajax)

$("#sendMail").on("click", function() {
var name = $("#name")
.val()
.trim();
var phone = $("#phone")
.val()
.trim();
var email = $("#email")
.val()
.trim();
var message = $("#message")
.val()
.trim();

if (name == "") {
alert("Please, write your name!");
return false;
} else if (phone == "") {
alert("Please, write your contact number!");
return false;
} else if (email == "") {
alert("Please, write your e-mail adress!");
return false;
} else if (message == "") {
alert("Please, write your message!");
return false;
}

$.ajax({
url: 'ajax/mail.php',
type: 'POST,
cache: false,
data: { 'name': name, 'phone': phone, 'email': email, 'message': message },
dataType: "html",
beforeSend: function() {
$("sendMail").prop("disabled", true);
},
success: function(data) {
if(!data) {
alert("Something wrong with internet connection! Please, try again!")
}
else {
$("mailForm").trigger("reset");
alert("Thanks for the application! we will contact you soon!");
}
$("sendMail").prop("disabled", false);
}
});
});
